# Árbol de Estudiantes en Java

Aplicación en consola que implementa un Árbol Binario de Búsqueda (ABB).

## Funcionalidades

- Insertar estudiantes (sin duplicados)
- Buscar estudiante
- Eliminar estudiante
- Mostrar estudiantes en orden
- Contar estudiantes
- Calcular altura del árbol

## Tecnologías

- Java
- Estructuras de datos
- Programación orientada a objetos

## Ejecución

javac Main.java  
java Main
